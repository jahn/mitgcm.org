Hardware/software requirements

MITgcm has been designed to run efficiently on a wide range of computer platforms. This is made possible by its WRAPPER interface which largely insulates the 'under the hood stuff' from the user.

MITgcm also runs on a single PC and PC clusters.

The minimum you require to run the model is a PC running linux and a FORTRAN compiler.
