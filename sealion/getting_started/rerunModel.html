Changing and re-running the model

Edit the file data 
% vi data
and change the line that says 
nTimeSteps=10,
to 
nTimeSteps=432,
The time-step is 600 seconds, so this will integrate the model for 10 days. Hopefully, it will finish well before that. ;^)

Re-run the model: 

% ./mitgcmuv > output.txt
When the model finishes, you'll notice more output files:
