Getting Started

Once you have obtained the code you can easily compile, run and examnine the model following the steps outlined on the left pane.

Of course, the are some basic requirements that you need to be able to use the model.

  • a UNIX-like OS
    • any of Linux, Solaris, IRIX
  • a fortran compiler
    • the native compilers for Sun, DEC, SGI
    • the free GNU compiler, g77
    • the Compaq compiler, fortran
    • the Interl compiler
  • disk space
    • the source itself is
  • a visualize tool
    • Matlab
    • Ferret
    • IDL, etc...
Now your ready to compile an example experiment...
